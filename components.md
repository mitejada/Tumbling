# Component Structure

**HOMEPAGE**
* SIGNUP PAGE
* LOGIN PAGE
* SEARCH BAR
* LOGO

**USER**
* USER PROFILE - DISPLAY POST, LIKES, FOLLOWING, ARCHIVE BAR
* USER ACCOUNT - POST AMOUNT, FOLLOWERS AMOUNT
* POST BAR
* SEARCH BAR
* DASHBOARD BAR
* LOGO

**DASHBOARD PAGE**
* NAVBAR
* SEARCH BAR
* ABILITY TO POST A POST VIA POST BAR
* EDIT POST
* POSTS AVATAR
* DISPLAY POSTS OF USERS FOLLOWINGS

**NAVBAR**
* DASHBOARD NAVBAR
* POSTING NAVBAR

**HEADER**
* SEARCH BAR
* DASH NAV BAR
* LOGO
